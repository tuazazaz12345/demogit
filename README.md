Dự án website bán sách là một ứng dụng web thương mại điện tử chuyên nghiệp dành riêng cho việc mua sách được xây dựng bằng kiến ​​trúc MVC sử dụng ngôn ngữ PHP thuần túy. 
Các công nghệ được sử dụng
Ngôn ngữ: PHP 8.x
Cơ sở dữ liệu: MySQL
Kiến trúc: Mô hình-Xem-Bộ điều khiển (MVC)
Giao diện người dùng: HTML5, CSS3, JavaScript (Vanilla JS), Bootstrap
Thư viện/Tích hợp:
PHPMailer: Ứng dụng gửi email xác thực, mã OTP và thông báo.
VNPay: Cổng thanh toán trực tuyến tích hợp.
Chatbot: Hệ thống hỗ trợ khách hàng tự động.
ngrok: Được sử dụng để cho phép truy cập internet vào trang web trong quá trình phát triển (đặc biệt cần thiết cho VNPay và Webhooks).
